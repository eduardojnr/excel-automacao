Regras para o funcionamento:

1° - <./0 Base> É a base vinda do robô que se deseja filtrar, deve conter o formato .xlsx. # Por hora, usar apenas uma base robô por vez!

2° - <./1 Pre Enriquecimento> É a base após os filtros iniciais

3° - <./2 Enriquecimento Inicial> São as bases após o enriquecimento no banco de dados interno, separadas por banco e Encontrados/Ñ Encontrados

4° - <./3 Enriquecimento Final> É para onde virá as bases após enriquecimento na NovaVida e novo enriquecimento no banco de dados interno

5° - <./Arquivo Discadora> É para onde seguirá a base final já no formato da discadora #Sem remover clientes da casa ou que já estejam na discadora
